# Disease Prediction System (Heart Disease)

## 📌 Project Overview
A classification system to assist in medical diagnostics by predicting the presence of heart disease based on clinical patient records.

## 🛠️ Key Algorithms
* Logistic Regression
* Support Vector Machine (SVM)
* Random Forest Classifier

## 📊 Evaluation
* **Metric:** Accuracy, Precision, Recall, and F1-Score.
* **Focus:** Tuned for high Recall to ensure high-risk cases are flagged immediately for clinical review.
