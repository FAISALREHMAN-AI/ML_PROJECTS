# Customer Segmentation Using K-Means

## 📌 Project Overview
An unsupervised learning project designed to categorize mall customers into 5 distinct groups based on annual income and spending scores. This helps retail businesses design personalized marketing campaigns.

## 🛠️ Technical Stack
* **Algorithm:** K-Means Clustering
* **Logic:** Elbow Method for optimal K-selection
* **Visualization:** Matplotlib, Seaborn

## 📊 Key Clusters
1. **Target Customers:** High Income, High Spend.
2. **Careful Customers:** High Income, Low Spend.
3. **Careless Customers:** Low Income, High Spend.
4. **Standard:** Average Income, Average Spend.
5. **Sensible:** Low Income, Low Spend.
