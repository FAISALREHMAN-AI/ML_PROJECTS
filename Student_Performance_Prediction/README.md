# Student Performance Prediction System

## 📌 Project Overview
This regression project predicts a student's final academic score based on demographic and behavioral features such as reading, writing, and study time.

## 🛠️ Technical Stack
* **Models:** Linear Regression, Random Forest Regressor
* **Evaluation:** R² Score (Coefficient of Determination)

## 📊 Results
* **Performance:** Achieved an R² Score of 0.87, indicating that 87% of academic performance variance is captured by the model features.
